## Architecture JEE & Systèmes Distribués
Cours de Prof. Mohamed YOUSSFI — ENSET Mohammedia

Ce dépôt rassemble 3 projets pratiques (TP) conçus pour maîtriser les fondations des architectures JEE et de l'écosystème Spring Boot, développés étape par étape.

# 🛠️ Sommaire des Projets
# TP1 — Injection des Dépendances
Objectif : Comprendre le couplage faible et l'inversion de contrôle (IoC).

Au programme : Instanciation dynamique, configuration par annotations et par fichier XML, et fonctionnement interne du moteur de Spring.

# TP2 — ORM, JPA & Spring Data
Objectif : Gérer la persistance des données sans écrire de SQL à la main.

Au programme : Mapping objet-relationnel (JPA/Hibernate), création d'entités, manipulation des repositories Spring Data, et transition d'une base de test H2 vers MySQL.

# TP3 — Application Web Spring Boot
Objectif : Développer une application web complète, sécurisée et connectée.

Au programme : Architecture MVC, rendu dynamique avec Thymeleaf, design avec Bootstrap 5, et sécurisation des accès avec Spring Security.

# 🧰 Technologies Utilisées
Backend : Java 17+, Spring Boot 3.2, Spring Data JPA, Spring Security

Frontend : Thymeleaf, Bootstrap 5

Bases de données : H2 (In-Memory), MySQL

Build : Maven
